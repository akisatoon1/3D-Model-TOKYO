## プロジェクト概要
電子地図データを適切に処理して、東京ドーム周辺の都市空間を仮想世界で表現する。

### 対象の電子地図データの特徴
- 約1m間隔の格子状標高値
- その場所を上空から見たときの最上位点の標高が記録されている
- 計測できていない地点もある
- 一定の範囲ごとの複数ファイルに分割されている

[詳しいデータ仕様はこちら](result/data_spec.pdf)。
